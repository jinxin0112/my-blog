# 谈谈vue双向绑定

> 你对vue的源码了解吗，给我讲讲双向绑定原理怎么实现的，详细描述什么时候监听变化的，什么时候触发变化的？

## 我的理解

vue在实例对象vue时，会将data里的数据绑定监听，具体的实现是用Object.defineProperty去达到数据劫持，当数据发生变化时，触发set方法去更新视图。
